# Java Algorithms Projects

This repository contains three Java example programs demonstrating algorithmic concepts:

- `alphaBetaPruning.java`
  - Implements the alpha-beta pruning algorithm on a small game tree.
  - Prints the optimal minimax value and reports whenever pruning occurs.

- `hiddenMarkovModel.java`
  - Computes the forward probability for a simple hidden Markov model.
  - Uses fixed state transition, observation, and initial probability matrices.

- `ticTacToe.java`
  - Implements a Tic-Tac-Toe game with a minimax-powered computer opponent.
  - Accepts user input for moves and plays optimally as the computer.

## Requirements

- Java JDK 8 or later

## Run

From the project directory, compile all files:

```bash
javac *.java
```

Run any program:

```bash
java alphaBetaPruning
```

```bash
java hiddenMarkovModel
```

```bash
java ticTacToe
```
